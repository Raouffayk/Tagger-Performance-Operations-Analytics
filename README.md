# Tagger Performance & Operations Analytics

## Overview
This project analyzes the performance of 120+ sports data taggers across 600+ global football competitions.

The goal was to support business decisions by tracking KPIs such as productivity, rating, and task efficiency.

## Business Problem
Operations management needed visibility into team performance and efficiency to identify underperformance and improve decision-making.

## Solution
- Extracted and cleaned data using SQL
- Built dashboards in Looker Studio
- Defined KPIs for performance tracking
- Analyzed trends and outliers

## Key Insights
- Identified underperforming taggers
- Detected performance gaps across teams
- Highlighted efficiency trends over time

## Tools Used
- SQL (Redash)
- Looker Studio
- Google Sheets

## Business Analysis Concepts
- KPI Definition
- Performance Analysis
- Data-driven decision support
